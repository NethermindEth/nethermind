    - 8
    - Vista
    - 7

    - SUSE Enterprise Linux (SLES)	12 SP2+	x64 (not tested by us)
    - Alpine Linux	3.10+	x64, ARM64
    - Raspberry PI4 ARM64, ARM32
    - Gentoo Linux ARM64
